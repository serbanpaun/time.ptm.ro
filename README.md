# Current Time 
Displays current time in 2 timezones: user's timezone and server's timezone.  
Each one can be configured to display your preferred timezone.

Default time zone is `Europe/Bucharest`.

> [!Important]
> Make sure you update Google Tag with your own G-TAG, in order to see Analytics.
> If you use other analytics tool (or none at all), just remove the entire code included starting with `<! Google tag (gtag.js) -->` and before `</head>`.
